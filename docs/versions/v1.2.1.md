# rediscc v1.2.1

**Release date**: 2026-04-07
**Previous version**: v1.2.0

## Summary

This release improves debug log clarity and adds comprehensive project documentation.

## Changes

### Debug log prefix update

The debug log prefix was renamed from `[LOG]` to `[Rediscc Log]` across all methods in `RedisDataStore`. This makes it easier to identify rediscc logs when the library is used alongside other packages that also write to stdout.

**Affected methods**: `Publish`, `Get`, `GetRaw`, `Set`, `Del`, `Keys`.

```go
// Before
fmt.Println("[LOG] Get", key)

// After
fmt.Println("[Rediscc Log] Get", key)
```

### Project documentation

Added two documentation files:

| `docs/project-overview.md` | Non-technical overview: what the project does, how it works, who uses it, security, and integrations |
